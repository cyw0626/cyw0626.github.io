JavaScript原型模式

发表于 2019-12-03 | 分类于 code | 次阅读

技术博主🕵️‍♂️上线ing…

在看ES6类的继承中，实例这个词反复出现，搞晕了都，于是查了一波。

实例

定义：实例是类的具象化产品。

  • 实例通过构造函数创建。实例创造出来就具有constructor属性(指向构造函数)和_proto_属性(指向原型对象)。

  • 构造函数中有一个prototype属性。这个属性是一个指针，指向它的原型对象。

  • 原型对象内部指针(constructor)指向构造函数。

原型链

定义：JavaScript只有对象这一种结构。在JavaScript中，每个对象都有一个指向它原型对象的内部链接。这个原型对象又有自己的原型，直到某个对象的原型为null为止，组成原型链。

  • 使用isPrototypeOf()/Object.getPrototypeOf()检查对象间是否是原型关系

    函数

    函数中this的指向

  • 函数调用模式，this指向全局对象window

  • 方法调用模式，this指向当前对象

  • 构造器模式，this指向新创建的实例对象

  • apply\call模式，这两个的第一参数即this，若第一个参数为null,this指向window;若第一个参数为一个对象，this就指向这个对象

构造函数与普通函数区别

  • 构造函数用new调用
  • 构造函数命名是首字母大写，普通函数驼峰式命名
  • 构造函数内部可以使用this关键字
  • 普通函数可以用return返回
